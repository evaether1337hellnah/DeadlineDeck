\# Deadline Deck



\*\*Deadline Deck\*\* — приложение для учета учебных заданий и дедлайнов.



Проект сделан на Python с использованием ООП и библиотеки Tkinter.



\## Что умеет приложение



\- регистрация и вход в аккаунт;

\- отдельные задания для каждого пользователя;

\- добавление заданий;

\- редактирование заданий;

\- удаление заданий;

\- смена статуса;

\- поиск по заданиям;

\- фильтр по статусу;

\- сортировка по дате, приоритету, статусу, предмету и названию;

\- экспорт таблицы в Excel;

\- светлая и темная тема;

\- сохранение данных в JSON;

\- защита пользовательских данных.



\## Используемые технологии



\- Python

\- Tkinter

\- JSON

\- hashlib

\- base64

\- Pillow

\- openpyxl
